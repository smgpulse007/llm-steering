# research archive

This folder stores the source-backed research gathered for this workspace.

## Files

- `steering_vectors_literature_review.md` - detailed overview of steering vectors, activation engineering, representation engineering, and SAE-based steering.
- `gemma4_runtime_notes.md` - official Gemma 4 runtime, model-size, and deployment notes relevant to local experiments.
- `2026-06-14_diffusiongemma_qwen_ui_research_notes.md` - handoff research notes for DiffusionGemma, Qwen3.6/Qwen3-Coder-Next, steering reliability, and the planned UI/refactor.
- `verified_sources.json` - structured list of verified papers, docs, and repos used to build the research summary.

## Curation policy

Sources in this folder were selected because they are one of the following:

- official Gemma / Hugging Face / Ollama documentation
- primary research papers
- official paper code repositories

The goal is not "every paper ever written," but a high-signal, source-verified working archive that is good enough to build experiments from without guesswork.
